Welcome to the mzlmachines project site. Use this page to share release
notes, showcase demos, or link to your Svelte application.

- Update `_config.yml` with your GitHub username and repository baseurl.
- Add posts under `_posts/` or standalone pages under the `docs` folder.
- Publish by enabling GitHub Pages to serve from the `/docs` folder.
